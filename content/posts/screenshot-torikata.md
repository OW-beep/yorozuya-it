## スクリーンショットとは

スクリーンショットとは、パソコンやスマホの画面に表示されている内容を、そのまま画像として保存する機能のことです。エラーメッセージを人に伝えたい時や、Webページの内容を記録しておきたい時など、幅広い場面で活躍します。端末によって操作方法が異なるため、ここで一度まとめて整理しておきましょう。

## Windowsでのスクリーンショットの撮り方

| 操作方法 | 撮影される範囲 |
|---|---|
| PrintScreenキーのみ | 画面全体をコピー(クリップボードのみ、ファイル保存はされない) |
| Windowsキー + PrintScreen | 画面全体を画像として自動保存 |
| Windowsキー + Shift + S | 好きな範囲を選んで切り取り |
| Alt + PrintScreen | 今アクティブになっているウィンドウのみ |

「Windowsキー + Shift + S」は、範囲を自由に選べるため、特定の部分だけを共有したい時に便利です。撮影後は画面右下に通知が表示され、そこから直接編集することもできます。

## Macでのスクリーンショットの撮り方

| 操作方法 | 撮影される範囲 |
|---|---|
| Shift + Command + 3 | 画面全体 |
| Shift + Command + 4 | 好きな範囲を選んで切り取り |
| Shift + Command + 4、その後スペースキー | 特定のウィンドウのみ |

Macでは初期設定で、撮影した画像がデスクトップに保存されます。保存場所を変えたい場合は、「Shift + Command + 5」で表示されるメニューから、保存先を変更することができます。

## スマホでのスクリーンショットの撮り方

- **iPhone(Face ID搭載モデル)**:サイドボタンと音量を上げるボタンを同時に押す
- **iPhone(ホームボタン搭載モデル)**:ホームボタンとサイドボタン(または上部ボタン)を同時に押す
- **Android**:電源ボタンと音量を下げるボタンを同時に押す(機種により多少異なる場合があります)

多くのAndroid機種では、通知パネルや設定メニューからも撮影できる場合があります。うまくいかない場合は、機種名と合わせて操作方法を確認するのが確実です。

## 撮影後にすぐ編集したい場合

WindowsもMacもスマホも、撮影直後に画面の隅にプレビューが表示されるのが一般的です。このプレビューをタップ(またはクリック)すると、そのまま矢印や文字を書き込む編集画面に移動できるため、エラー箇所を丸で囲んで人に共有する、といった使い方がしやすくなっています。

## まとめ

スクリーンショットの撮り方は端末ごとに異なりますが、「画面全体」「好きな範囲だけ」の2パターンを押さえておけば、ほとんどの場面に対応できます。よく使う操作は、一度覚えてしまえば指が自然に動くようになるので、ぜひこの機会に試してみてください。
